\# AI Fire and Human Detection System 🔥👤



Real-time fire and human detection system built using \*\*YOLOv9, Flask, and OpenCV\*\*.



\## Features

\- Real-time camera monitoring

\- Fire detection using AI

\- Human detection

\- Email alert system

\- SMS alerts using Twilio

\- Live dashboard

\- Event logs

\- Snapshot capture



\## Tech Stack

\- Python

\- YOLOv9

\- Flask

\- OpenCV

\- Twilio

\- SMTP Email

\- HTML/CSS Dashboard
